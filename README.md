# Political Website

Minimal PHP political website starter.

Setup
- Copy `.env.example` to `.env` and fill values.
- Place project in your webroot (XAMPP: `c:/xampp/htdocs/Psite`).
- Start Apache & MySQL and adjust `config/db.php` or `.env` as needed.

Security notes
- Keep `.env` out of version control and do not expose `config/` to the web.
